# GUEST

This repository summarizes some additional functions used in the manuscript entitled "GUEST: An R package for handling estimation of graphical
structure and multi-classification for error-prone gene expression" written by Li-Pang Chen and Hui-Shan Tsao.

The function "spe_sen_bias_kl" in the R script spe_sen_bias_kl.R provides values of specificity (SPE) and sensitivity (SEN) and is used to examine the performance of the estimators via simulation studies.
